# 📝 PasteBin in Go!

A tiny pastebin clone built *entirely* with Go’s standard library — no frameworks, no nonsense.

## ⚙️ Run it

```bash
git clone https://github.com/your-username/go-pastebin.git
cd go-pastebin
go run main.go
```
Then visit: http://localhost:8080
## 🎯 Features

- Create & share text pastes
- Clean UI
- Blazing fast
